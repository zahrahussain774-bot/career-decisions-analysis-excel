# Career Decisions Analysis in Microsoft Excel

## Project Overview

This project analyzes career decision data using Microsoft Excel. The analysis focuses on identifying patterns and trends that influence career choices through data cleaning, formulas, pivot tables, and visualizations.

## Dataset

The dataset contains information related to career decisions and factors that may influence career preferences and outcomes.

## Objectives

- Clean and organize the dataset.
- Explore factors influencing career decisions.
- Perform descriptive analysis.
- Summarize findings using Pivot Tables.
- Create charts to visualize trends and patterns.
- Generate meaningful insights to support decision-making.

## Excel Skills Used

- Data Cleaning
- Sorting & Filtering
- Conditional Formatting
- IF Functions
- SUM, AVERAGE, MIN, MAX
- COUNTIF / SUMIF
- Pivot Tables
- Charts
- Data Validation

## Tools Used

- Microsoft Excel

## Key Findings

- Organized and cleaned the dataset for analysis.
- Identified trends and patterns in career-related data.
- Created Pivot Tables to summarize information.
- Developed charts to present findings visually.
- Generated insights to support career decision analysis.
